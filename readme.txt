The BatchAnalysis-generic.xlsm spreadsheet is designed to be an extension of the Find-MailboxDelegates script at https://aka.ms/find-mailboxdelegates on GitHub. 

There is a blog at http://blogs.technet.microsoft.com/fasttracktips titled "Who is linked to whom in your spider web of mailbox permissions. Migration batch permissions analysis." that outlines using the script output with the spreadsheet.
